# 🚀 SC Database — Star Citizen Database

[![Ko-fi](https://img.shields.io/badge/Ko--fi-Support%20the%20project-FF5E5B?logo=ko-fi)](https://ko-fi.com/tubsdep)
[![Live Site](https://img.shields.io/badge/Live-Site-4fc3f7)](https://production.sc-database-a7j.pages.dev/)

Base de datos completa e interactiva de **Star Citizen** con información detallada sobre misiones, blueprints, minerales, items, armas de nave, componentes y contratos Wikelo. Todo en una interfaz rápida, buscable y responsive.

## 🌐 Live Site

[https://production.sc-database-a7j.pages.dev/](https://production.sc-database-a7j.pages.dev/)

## ✨ Features

- **📋 Misiones** — 1,786 misiones filtrables por facción, sistema, tipo, legalidad y blueprints
- **🔧 Blueprints** — 1,591 planos de fabricación categorizados por tipo (componentes, armas, armaduras)
- **💎 Minerales** — Todos los minerales con rareza, firmas (signature) y locaciones por sistema
- **🔫 Armas de nave** — 137 armas con stats detalladas (DPS, alpha, range, firerate, precio)
- **📦 Items** — 7,489 items clasificados en 18 categorías (armaduras, ropa, armas FPS, comida, etc.)
- **🛸 Wikelo** — Contratos de trueque con inputs/rewards
- **⚙️ Componentes** — Power plants, shields, coolers, quantum drives con size y grade
- **🏛️ Facciones** — Organizaciones y misioneros de Stanton y Pyro

## 🎨 Temas

- **Stanton** — Tema oscuro con acentos azules
- **Pyro** — Tema hostil con acentos naranja/rojo y efectos glitch/CRT

## 🛠️ Stack

- **Frontend:** HTML + CSS + JavaScript vanilla
- **Backend:** Python + Flask API
- **Hosting:** Cloudflare Pages + GCP
- **Datos:** JSON procesados desde datos scrapeados del juego
